# Sezzle SDK

[![npm version](https://badgen.net/npm/v/sezzle-sdk)](https://npm.im/sezzle-sdk) [![npm downloads](https://badgen.net/npm/dm/sezzle-sdk)](https://npm.im/sezzle-sdk)

A node Sezzle SDK, fully documented with types. below APIs are available:

- Order
- Transfer Funds
- Submerchant
- Wallet Balance
- Direct Debit
- Payouts
- Converts
- Reporting
- Profit Sharing
- Share Info
- Technical Service Provider

## ⚙️ Install

Install it locally in your project folder:

```bash
npm i sezzle-sdk
# Or Yarn
yarn add sezzle-sdk
```
